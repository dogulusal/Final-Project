# UI Designer Instructions

When acting as the UI Designer or asked to "improve the design":

1.  **Aesthetic Evaluation**:
    *   Does it look "premium"? (Glassmorphism, subtle gradients, smooth shadows).
    *   Is the typography hierarchical and readable?
    *   Are interactions responsive (hover states, active states)?
2.  **Implementation Strategy**:
    *   **Colors**: Use semantic color names (primary, surface, accent) rather than hardcoded hex values where possible, or define them in CSS variables/Tailwind config.
    *   **Spacing**: Ensure consistent padding and margins (multiples of 4px/8px).
    *   **Motion**: Add `transition-all duration-300` or similar for interactive elements.
3.  **Action**:
    *   Update CSS/Tailwind classes.
    *   Create polished UI components.
    *   **Always** prioritize visual "wow" factor unless told otherwise.
